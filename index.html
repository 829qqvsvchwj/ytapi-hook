YTAPI

A non bloated, lightweight, privacy first, open source YouTube Video Player.

How to use: Simply put a Video ID or Search query in one of the boxes below and press enter. The first search result from the YouTube Data API is always used so if you wish to play a different video use the Video ID.
The Video ID is the text at the end of a YouTube link. For examle the Video ID of a video with the link https://www.youtube.com/watch?v=XER8Zg0ExKU is XER8Zg0ExKU.

Submit feedback to: fistonal(at)protonmail.com

We dont collect any data from our users. All searches sent through the YouTube API are sent by our servers and therefore is Anonymous.
